I. Minimum Requirements

Windows

A PC with specs made in 2013 and later is recommended.

    OS: Windows 7 SP1
    Processor: One with 2.4+ GHz
    Storage: 474 MB required for Half-Life; 1.5+ GB required for ZAMNHLMP.

macOS

    Half-Life: Zombies Ate My Neighbours and the Aura engine no longer support macOS.

Linux

A PC with specs made in 2010 and later is recommended.

    OS: Any distro that Steam can run on
    Processor: One with 2+ GHz
    Storage: 474 MB required for Half-Life; 1.5+ GB required for ZAMNHLMP.

II. General Game Issues

Game Crashes

Game crashes with Metamod installed

Don't install the ordinary Metamod. Instead, install the one from here. (NOTE: This version of Metamod does not work with ordinary listen or dedicated servers. You will need ReHLDS for it to work.

LAN Heartbeats

By default, the Half-Life game engine now reports the current status of IP LAN (e.g., sv_lan 1 or unable to authenticate) games to the master servers. This reporting is for general statistical information to Valve only and these IP LAN servers will not be visible to Internet players. If you would like to opt out of having your IP lan server report to the master servers, you must run the engine or dedicated server with the "-nomaster" command line option.

Joystick Functionality

The Joystick Functionality in the Half-Life engine is very outdated and only works with joysticks made in the late '90s and early '00s. If you wish to use a controller for playing ZAMNHLMP, you can use the following software solutions:

  • Steam Big Picture Mode: Big Picture Mode on Steam has a functionality where you can make the controller emulate a keyboard & mouse.
  • Controller Companion: This is a software that Sabian Roberts, the lead developer of The Phoenix Project, uses himself. This software also allows you to emulate a keyboard & mouse with a controller; however, it is paid software and only works with XInput (Xbox, Xbox 360 and Xbox One controllers). Unlike Steam Big Picture mode, it allows you to use your controller on any other program such as Google Chrome or Microsoft Word [using its on-screen keyboard].

III. Frequently Asked Questions

Why does the gauss/tau cannon not fire, but the ammo goes down in some multiplayer games?

The answer here is ping latency. Try to improve your ping by moving closer to your router, using a wired connection etc., restarting your computer, restarting Steam, playing on a different server or using Ping Enhancement Software.

What is the difference between this game and Adrenaline Gamer?

Quite a few. For example, there is a massive set of maps, easier accessible menus, more and modified gamemodes, reanimated playermodels, viewroll, viewmodel bob tilt, and a few extra goodies.

Why can I see incomplete parts of the weapons' viewmodels?

Simple answer: FOV. You can change the FOV of the firstperson view by opening the console (set the control in Keyboard settings), typing default_fov in the command line and then the value of your choice. The minimum is 20 and the maximum is 150. The default value is 110. Though it is declared in the userconfig.cfg file, the value is also hard-coded into the game.
